# Requirements for the project

## Application properties

* $APP_NAME: app name (site title, etc);
* $APP_DESC: app description (homepage, manifest);
* favicon (ICO);
* PWA manifest:
    * background_color
    * description
    * icons (PNG, 180x180, 192x192, 512x512)
    * name
    * short_name
    * theme_color
